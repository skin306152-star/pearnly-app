## 锁定后的分阶段(覆盖 docs/00 默认切法)

- **Phase 1 · 合规开票闭环**:商品主数据(内建+Excel 导入+**图**)→ 选客户 → 菜单图卡/手输码加行 → 7% VAT
  → 生成合规 ใบกำกับภาษี/ใบเสร็จ PDF(**连号锁 + 开出不可改 + 红冲/补开**)→ 发送(邮件/LINE/打印)。
- **Phase 2 · 库存 + LINE 开单 + 应收**:`inventory_*` 库存扣减/查询 · LINE bot 开单 · 应收追踪(账龄/催收)。
- **Phase 3 · e-Tax 直报**:对接 RD e-Tax Invoice & e-Receipt(需电子证书 · "可行才做")。
- **下一个独立项目 · POS**:收银/扫条码/扫 QR/现场卖货(不在本模块)。

## (存档)原分支分析 · 客户回前的每答案影响

## Q1 · POS 定位

| 答案 | 设计影响 |
|---|---|
| (1) 只是快速开票 | **默认路线**。无库存/收银表。products 只做主数据。工作量可控。 |
| (2) 真零售 POS | 新增 `inventory_*`/`pos_sessions`/现金抽屉,接小票打印机/扫码枪硬件。独立大模块,Phase 拉长。 |
| (3) 先开票后扩 POS | 走(1),但 products 预留库存/收银字段,模型不返工。 |

## Q2 · "从照片选品"含义

| 答案 | 设计影响 |
|---|---|
| (1) 拍照 AI 识别商品 | 复用 Gemini,extract 端返回候选商品。`products.image_url` 非必需。 |
| (2) 商品图库点选 | products 必须带图(`image_url` 必填路径 + 上传)。前端做菜单网格。 |
| (3) 拍旧票反向开单 | 复用 OCR 把旧单据转销项草稿。和进项 OCR 同管线。 |

## Q3 · 合规级别(决定 MVP 硬不硬)

| 答案 | 设计影响 |
|---|---|
| (1) 合规 PDF 即可 | MVP:连号 + 双方税号 + 7% VAT 分列。红冲/e-Tax 后置。 |
| (2) 连号锁 + 不可改 + 红冲 MVP 就上 | 立即落 `document_number_sequences` 严格取号 + issued 写保护 + `credit_debit_notes`。 |
| (3) 全套含 e-Tax 上报 | 加 `etax_submissions` + 对接 RD e-Tax(需数字证书)。最重。 |

## Q4 · 商品主数据来源

| 答案 | 设计影响 |
|---|---|
| (1) 手建 + Excel 导入 | 加 `/api/products/import` + `product_imports` 表 + Excel 解析。 |
| (2) Mr.ERP 同步 | 需确认 Mr.ERP 有商品 API;加同步任务。复用现有 ERP 集成。 |
| (3) 暂不要主数据,现填 | products 表暂缓,行项目允许纯文本现填。最快出 MVP。 |

## 附加 · 客户用 Paypers 最痛的点(决定第一刀"智能"砸哪)

- 录入慢? → 先做 LINE/拍照极速录入
- 查税号麻烦 / 易填错被税局退? → 先做 RD 验真 + 规则校错
- 不能对账 / 不能进 ERP? → 先做对账核销 + 一键推 ERP
- **没有客户原话就别猜**,问卷末尾已设这道开放题。

## 补充细节 → 全做兼容/可配置 · 无需问客户(Zihao 2026-06-05 拍板)

不再做第二份客户问卷;这几条全在系统里兼容掉,带安全默认,客户建档时按需调:

| 细节 | 处理 |
|---|---|
| **连号规则**(按月/按年重置·册号·前缀) | **可配置 + 默认兜底**:默认 `前缀-年份-连续号` 且一直累加不跳号(永远合规)· 客户可改前缀/重置周期/位数/册号/**起始号(接旧账本)**。法律只要求"连续不跳/不重",格式公司自定(§86/4)。见 `docs/02` document_number_sequences。 |
| **卖方是否注册 VAT** | 每客户 `vat_registered` 开关:注册→开税票(含7%),未注册→只开收据。两种都支持。 |
| **发送渠道** | 邮件/LINE/打印**全做** · `default_send_channel` 默认 email,可改。 |
| **商品图来源** | 手动上传 + Excel 导入带图,**两种都做**。 |
| 主要用户 | 已答:事务所 + 客户公司自开 **都支持**(复用多租户)。 |

> 唯一可能需客户填的:建档时"有没有旧纸质账本起始号要接"——设置里一个选填项,非问卷。
