## 调研补充 · 还能让用户测的(Owner 可挑)
- **置信度分层体感**:同一票清晰版 vs 模糊版 → 看高置信直推 / 中置信问一句 / 低置信留人工。
- **语言**:全泰文 vs 中泰混排(若有) → OCR 语言鲁棒。
- **同供应商多票**:测供应商归一(不同写法同一家)。
- **跨账期日期**:落在已结期的日期 → 看是否拦/红冲提示(若 Express 账期生效)。
- **税务类型**:แยก VAT(税分列) vs รวม VAT(含税) 各一张。
- **大额 vs 小额**:百万级 vs 几十铢 → 金额范围鲁棒。
