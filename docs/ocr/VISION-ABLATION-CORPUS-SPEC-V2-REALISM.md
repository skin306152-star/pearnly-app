- 泰文字体 Sarabun / Noto Sans Thai;热敏点阵风。

**README 自检清单(逐项打勾,抽检 10 张附图)**:

## 二、对账类:描真件 + 公开样本再生(补银行版式多样性)

不再用「想象的」银行版式。改为两条来源:

1. **描真件**:读给定的真实对账样本(项目/桌面提供:TTB 对账单+GL、真 ภาษีขาย 销项税报告、
   真 bank GL+statement),提取其**真实版式**(列位置 / 页头页脚 / 字体 / 堆叠方式 / 借贷余额列排布),
   做成**仿该银行的模板渲染器**。

2. **公开样本参照(铺开更多银行)**:web 搜各泰国银行的**官方样板 / 空白模板 / 格式说明**,覆盖
   **KBank / SCB / BBL / Krungsri(BAY) / TTB / GSB / KTB** 等主流行,学**版式**再生。

   🔴 **红线(必须守)**:只描**版式骨架**,数据一律填**我们自己的合成数据**。
   **严禁使用任何含真人真实财务数据的文件**(网上扒到的真实对账单 / 泄露件)——哪怕仅当参照也不行,
   隐私 + 法律风险。**只允许官方空白样板 / 示意图 / 格式文档**。拿不准的来源一律不用。

3. **规模**:银行对账单 ≥ 6 家行 × 各 ≥ 8 张变体(换数据 + 退化档),GL ≥ 12,VAT ≥ 12。
   每张 GT 逐字精确(合成数据我们全知道),勾稽自洽(sum 明细 = 汇总、借贷平、余额链连续)。
