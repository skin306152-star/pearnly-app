## คู่มือการใช้งานระบบ POS ร้านค้าปลีก / ร้านสะดวกซื้อ

### 1. ตั้งค่าหัวใบเสร็จ (ชื่อร้าน + ที่อยู่)

ใบเสร็จทุกใบจะพิมพ์ชื่อร้าน / ที่อยู่ / เลขประจำตัวผู้เสียภาษี / เบอร์โทร ที่ด้านบน — ข้อมูลนี้ดึงมาจาก
หน้า **「ข้อมูลบริษัท」** (เมนูซ้าย → 公司资料 / Company Profile)

**ขั้นตอนแก้ไข:**
1. เข้าเว็บหลัก (ไม่ใช่หน้าขายหน้าร้าน) → เมนูซ้าย → **公司资料**
2. เอาเมาส์ไปวางที่ช่องข้อมูล (企业名称 / 注册地址 / 联系电话 / 纳税人识别号) → กด **「编辑」** → พิมพ์ข้อมูล → บันทึก
3. เปลี่ยนแล้วมีผลทันที ใบเสร็จใบต่อไปจะพิมพ์ข้อมูลใหม่โดยอัตโนมัติ (ไม่ต้องรีสตาร์ทเครื่อง)

> **หมายเหตุ:** ถ้าเครื่องคิดเงินออฟไลน์ (ไม่มีเน็ต) จะพิมพ์ใบเสร็จแบบง่าย โดยใช้ชื่อร้าน+ที่อยู่ที่ดึงไว้ล่าสุด
> ตอนเครื่องต่อเน็ตครั้งก่อน — ถ้าเพิ่งแก้ข้อมูลร้าน ให้เปิดเน็ตแล้วเข้าหน้าขายรอบนึงก่อน ข้อมูลจะอัปเดต

### 2. ตั้งค่าการรับเงิน

เมนูซ้าย → **收银业务 → 收款设置**(หน้านี้เจ้าของร้านเท่านั้นที่แก้ได้ พนักงานแคชเชียร์เห็นแต่แก้ไม่ได้)

รองรับ 4 ช่องทาง:

| ช่องทาง | ปิด/เปิดได้ไหม | ต้องตั้งค่าอะไร |
|---|---|---|
| **เงินสด (เงินสด)** | เปิดตลอด แก้ไม่ได้ | ไม่ต้องตั้งค่า ระบบคำนวณเงินทอนอัตโนมัติ |
| **PromptPay สแกน** | เปิด/ปิดได้ | กรอกเบอร์โทร / เลขภาษี ที่ช่อง "收款 PromptPay ID" |
| **รูดบัตร** | เปิด/ปิดได้ | ไม่ต้องตั้งค่า(ตอนนี้เป็นโหมดบันทึกอย่างเดียว เงินเข้าบัญชีธนาคารผ่านเครื่องรูดบัตรของธนาคารโดยตรง) |
| **โอนเงินเข้าบัญชี** ★ใหม่ | เปิด/ปิดได้ (ปิดอยู่จนกว่าจะตั้งค่า) | กรอก **ชื่อธนาคาร / เลขที่บัญชี / ชื่อบัญชี** ทั้ง 3 ช่อง |

**เปิดใช้ "โอนเงินเข้าบัญชี":**
1. หน้า「收款设置」→ แถว "银行转账" → กดสวิตช์เปิด
2. กรอก 3 ช่อง: ชื่อธนาคาร(เช่น SCB) / เลขที่บัญชี / ชื่อบัญชี
3. กด **「保存」**
4. กลับไปหน้าขายหน้าร้าน จะเห็นแท็บ "โอนเงิน" (transfer) เพิ่มขึ้นมาในหน้ารับชำระ

**ตอนแคชเชียร์รับเงินโอน:**
1. กดปุ่ม "รับชำระ" → เลือกแท็บ "โอนเงิน"
2. หน้าจอจะโชว์ชื่อธนาคาร/เลขบัญชี/ชื่อบัญชีให้ลูกค้าดู(อ่านให้ลูกค้าฟังหรือหันจอให้ดูได้)
3. รอลูกค้าโอนเงินจริง → เช็คสลิป/แอปธนาคารว่าเงินเข้าแล้ว → กด **「ได้รับเงินโอนแล้ว · ออกใบเสร็จ」**
4. ใบเสร็จจะพิมพ์ "付款方式:转账" ไว้ด้วย

> ⚠️ ระบบยังไม่เชื่อมกับธนาคารอัตโนมัติ ต้องให้แคชเชียร์ **เช็คยอดเงินเข้าเองก่อนกดยืนยัน** ทุกครั้ง —
> เหมือนกับการรูดบัตร ระบบแค่บันทึกไว้ในรายงาน ไม่ได้ตัดเงินจริง

### 3. เชื่อมต่อ Google Sheet เก็บประวัติการขาย

เมนูซ้าย → **收银业务 → Google Sheet 留档**(เจ้าของร้านเท่านั้น)

ฟีเจอร์นี้จะเอาทุกบิลที่ขายได้ ไปเพิ่มเป็นแถวใหม่ใน Google Sheet ของร้านเองอัตโนมัติ ไว้เก็บประวัติดูย้อนหลัง
(ไม่ใช่บัญชีของ Pearnly — เป็นชีตในบัญชี Google ของเจ้าของร้านเอง)

**ขั้นตอนเชื่อมต่อ (ทำครั้งเดียว):**
1. เปิด Google Sheet ที่จะใช้เก็บยอดขาย (สร้างชีตใหม่เปล่า ๆ ก็ได้ ไม่ต้องมีหัวตาราง ระบบจะเติมให้)
2. คัดลอกลิงก์ของชีตนั้น(กด "แชร์" หรือคัดลอกจาก address bar)
3. กลับมาหน้า「Google Sheet 留档」→ กด **「连接 Google」** → ล็อกอินบัญชี Google ของร้าน → กดอนุญาต(Allow)
4. ระบบจะพากลับมาหน้าเดิม สถานะขึ้น "เชื่อมต่อแล้ว" พร้อมอีเมลบัญชีที่เชื่อม
5. วางลิงก์ชีตที่คัดลอกไว้ในช่อง "ชีตปลายทาง" → เปิดสวิตช์ "เปิดบันทึกอัตโนมัติ" → กด **「保存」**

**หลังเชื่อมต่อแล้ว:** ทุกครั้งที่ขายของสำเร็จ ระบบจะเพิ่มแถวใหม่ในชีตให้อัตโนมัติ(เลขที่ใบเสร็จ / เวลา /
รายการสินค้า / ยอดรวม / วิธีชำระ) — ไม่ต้องกดอะไรเพิ่ม ทำงานเบื้องหลังไม่รบกวนการขาย

**ปัญหาที่พบบ่อย:**
- **ขายแล้วแต่ชีตไม่ขึ้นแถวใหม่** → เช็ค 3 อย่าง:① สวิตช์ "เปิดบันทึกอัตโนมัติ" เปิดอยู่ไหม ② ลิงก์ชีตที่วางไว้ถูกต้องไหม
  ③ ยังเชื่อมต่อ Google อยู่ไหม(บางทีถอนสิทธิ์จากฝั่ง Google เอง ระบบจะขึ้น "ยังไม่ได้เชื่อมต่อ")
- **ไม่กระทบการขาย**:ต่อให้เชื่อมต่อ Google มีปัญหา(เน็ตล่ม/สิทธิ์หมด)ก็ยังขายของออกใบเสร็จได้ตามปกติ
  ระบบแค่ข้ามการบันทึกลงชีตรอบนั้นไปเงียบ ๆ ไม่มีข้อความ error โผล่มากวนแคชเชียร์
- **ต้องการยกเลิก** → กด **「断开」** ที่หน้าเดิม

### 4. ขั้นตอนขายประจำวัน

```
เริ่มกะ → เลือกสินค้าใส่ตะกร้า → กด「รับชำระ」→ เลือกวิธีจ่าย → ยืนยัน → พิมพ์ใบเสร็จ → (ทำซ้ำ) → ปิดกะ
```

1. **เริ่มกะ**:เข้าเครื่อง → ล็อกอินแคชเชียร์(เลือกชื่อ + ใส่ PIN) → กด **「เริ่มขาย」**(ใส่เงินทอนเริ่มต้นถ้ามี)
2. **ขายของ**:แตะสินค้าใส่ตะกร้า(หรือสแกนบาร์โค้ด) → ถ้าลูกค้าจะซื้อของอื่นก่อนออกไปหยิบเงิน
   ใช้ **「เรียกบิลพัก」** เก็บบิลไว้ก่อนได้ ไม่ตัดสต๊อก ไม่ออกใบเสร็จจนกว่าจะเรียกบิลกลับมาคิดเงิน
3. **รับเงิน**:กด **「รับชำระ」** → เลือกช่องทาง(เงินสด/PromptPay/รูดบัตร/โอนเงิน — เห็นเฉพาะช่องที่เจ้าของเปิดไว้)
   → ยืนยัน → ระบบตัดสต๊อกอัตโนมัติ + พิมพ์ใบเสร็จ
4. **ลูกค้าขอใบกำกับภาษีเต็มรูป** → กด **「开正式税票」** บนหน้าจอสำเร็จ → กรอกชื่อ/เลขภาษีผู้ซื้อ →
   ออกใบกำกับภาษี(⚠️ ออกแล้วแก้ไม่ได้ ต้องออกใบลดหนี้ถ้าผิด)
5. **คืนสินค้า** → เมนู **「คืนสินค้า」** → สแกนใบเสร็จเดิม → เลือกสินค้าที่คืน → ระบบคืนเงิน + เติมสต๊อกกลับอัตโนมัติ
6. **ปิดกะ / สรุปยอด**:ท้ายกะ → เมนูปิดกะ → ระบบโชว์ยอดขายรวม แยกตามช่องทาง(เงินสด/PromptPay/บัตร/โอนเงิน)
   → นับเงินสดในลิ้นชักจริง กรอกยอดที่นับได้ → ระบบเทียบส่วนต่าง → กด **「提交交班」** ออกรายงานสรุปวัน

### 5. แก้ปัญหาเบื้องต้น

| อาการ | สาเหตุที่พบบ่อย | วิธีแก้ |
|---|---|---|
| ไม่มีสัญญาณเน็ต | เน็ตร้านหลุด | ระบบสลับเป็นโหมดออฟไลน์อัตโนมัติ(มุมขวาบนขึ้น "离线中")ยังขายได้ต่อ บิลจะซิงก์เข้าระบบอัตโนมัติเมื่อเน็ตกลับมา |
| เครื่องพิมพ์ใบเสร็จไม่ทำงาน | เครื่องพิมพ์ไม่ต่อ / กระดาษหมด | ตรวจสายเครื่องพิมพ์ก่อน ถ้ายังไม่ได้กด "พิมพ์ใบเสร็จ" ซ้ำได้ที่ปุ่มเดิม ไม่เสียเงินซ้ำ |
| ยกเลิกรายการที่คิดเงินผิด | พนักงานกดผิด | ใช้ **「คืนสินค้า」** ย้อนหลัง(ไม่ใช่ลบบิล ทุกบิลที่ออกแล้วต้องมีร่องรอย เพื่อบัญชีถูกต้อง) |
| ยังไม่เห็นแท็บ "โอนเงิน" ทั้งที่เปิดสวิตช์แล้ว | ลืมกดบันทึกหลังกรอกข้อมูลธนาคาร | กลับไปหน้า「收款设置」เช็คว่ากด「保存」แล้วหรือยัง |
| ลูกค้าอยากได้ใบกำกับภาษี แต่ปุ่มกดไม่ได้ | บิลนี้เป็นบิลออฟไลน์(ยังไม่ซิงก์เข้าระบบ) | รอเน็ตกลับมา บิลซิงก์เสร็จแล้วค่อยเปิดประวัติมาออกใบกำกับภาษี |
