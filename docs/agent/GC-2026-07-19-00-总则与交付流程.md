## 2. 开工规矩（每个施工窗口第一件事）

1. **禁止在共享工作树（`C:\Users\skin3\Desktop\pearnly-app`）切分支/reset/rebase/stash。** 三个窗口共用它的 master。
2. 用隔离 worktree 施工：
   ```
   cd C:\Users\skin3\Desktop\pearnly-app
   git fetch origin
   git worktree add C:\Users\skin3\Desktop\gc0719-<a|bc> -b wip/gc0719-<a-entrance-token|bc-bank> origin/master
   ```
   worktree 建好后先确认 `git log -1` 就是 origin/master 头（血泪：worktree 基线滞后）。
3. 施工全程只在自己的 worktree 里进行；依赖安装照根仓库（node_modules 可用 junction 或重装，npm ci）。

## 3. 施工与提交规矩

- 只改施工单点名的文件；发现单里没写的问题 → **停手**，记进交付 REPORT 的「待拍板」节，不顺手修（铁律：只修派的）。
- commit 一律 `git add <逐个文件名>`，禁 `add -A`/`add -u`/裸 `git commit -a`（血泪：子代理不带 pathspec 吞 index）。
- Conventional Commits，小步提交；空跑 commit 必 `--only`。
- 改了 `src/*.js` → 同一个 commit 里必须带 `npm run build` 产物 `static/dist`（prod 不重建 dist）。改 `.html` 引用的资源 → bump `?v=`。
- 纯 `.py` / 文档 / 测试改动不需要 build。
- **绝不 push master。** 只 push 自己的 wip 分支：`git push origin wip/gc0719-...`。push master = 直接上生产，本批一律由验收窗口合入。
- 不绕 hook，不 `--no-verify`。

## 4. 自测硬门（每个交付点 push 分支前，全部本地跑绿）

```
python -m unittest discover -s tests -q          # 全量（5700+），不许挑子集
python -m black --check <改过的每个 .py>
python -m ruff check <改过的每个 .py>
npx prettier --check <改过的每个 js/json/css/html>   # 逐文件点名，别全仓扫
npx eslint <改过的每个 js>
python scripts/check_imports.py --quiet
python scripts/check_file_size.py
python scripts/check_line_ratchet.py
python scripts/check_ui_consistency.py           # 动了前端才需要
python scripts/check_theme_responsive.py --gate  # 动了前端才需要
python scripts/check_new_debt.py
python scripts/check_cachebust.py
```

- 测试框架只用 unittest，禁 pytest。每个新文件 ≥1 个对应测试。
- 本地 `DATABASE_URL` 若指向本地 docker，跑单测前确认没被污染（血泪：本地 env 污染单测）。
- 涉及 UI 的验收必须真浏览器 Playwright：`isVisible` + `getComputedStyle` + 截图，截图存 `tests/e2e/_artifacts/gc0719_<a|bc>/`。grep 类名 / 断言常量 = 不算验证。
- wip 分支 push 后 CI 会自动全跑（`branches: ["**"]`），**自己盯到全绿**；红了自己修，不留红交付。

## 5. 交付物（每张施工单一个交付点）

分支上最后一个 commit 必含交付报告 `docs/agent/GC-2026-07-19-REPORT-<A|B|C>.md`：

1. 改动清单（文件 + 一句话）；
2. 自测证据：unittest 汇总行、逐文件 lint 命令与结果、E2E 截图相对路径、CI run 链接（绿）；

5. 回滚方式（一句话：revert 哪几个 commit / 关哪个闸）。
