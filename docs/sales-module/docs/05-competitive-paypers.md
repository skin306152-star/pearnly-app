# 05 · 竞品对比:Paypers vs 我们("更智能"落点)

> Zihao 拍板:不止对标 Paypers,要**比它更智能**。下面把"更智能"落成具体能力,
> 每条都标我们**已有的零件**——证明这不是吹,是把现成能力接到新口上。

## Paypers 是什么(据客户 11 张截图)

Thai 记账 App(app.paypers.ai),截图均为"支出/รายจ่าย"录入:
- 录入靠**手填表单 + LINE 打字**("ค่าน้ำ 50" → 自动记一笔 + 回卡片)
- 按名称/税号搜对方、分类/子分类、VAT/WHT/四舍五入、付款状态、出 ใบแทนใบเสร็จ PDF
- 本质:**录得快的录入工具**(单机闭环,导出了事)

## 逐项对比

| 能力 | Paypers 现状 | 我们的"更智能" | 靠什么(已有) |
|---|---|---|---|
| 录入 | 手填 + LINE 打字 | LINE 发**图/语音/一句话** → Gemini 全自动提取;不清楚就**多轮反问**确认 | Gemini OCR 管线 |
| 对方信息 | 手搜名称/税号 | 输税号 → **直连税局**带 17 字段 + **验真伪** | `rd_routes` |
| 录入校验 | 基本必填 | 知识库规则引擎:算术/完整/**查重**,报税前拦错 | `services/knowledge` |
| 分类/科目 | 人工选 | AI 推荐分类 + 借贷科目分录 | 规划中凭证中心 |
| 对账 | ❌ 无 | 录完自动与**银行流水核销** | 对账中心 |
| 下游 | 独立导出 | 一键推 **Mr.ERP / Xero** | ERP 集成 |
| 销项选品 | (无销项) | 手输码/扫条码/扫QR/**拍照识别** + 菜单点选 | 本模块新建 + OCR |
| 语言 | 泰语为主 | **中/泰/英**(华人在泰做账客群) | 现有 i18n |

## 一句话定位

> Paypers = "录得快"。
> 我们 = "录得快 + 自动查真伪 + 自动算 + 自动校错 + 自动对账 + 自动入 ERP"——带 AI 的全链路。
