> 防"死按钮 / 假功能"。每个 UI 控件:点了做什么、调哪个接口(对齐 04)、什么状态可点、结果。
> **施工流程**:施工前此表必须全填(无"待定");施工后每行 E2E 真点一遍断言路由真通 + 打勾(✅施工后回填)。10-acceptance 据此验收。
> 列含义:能点条件 = 何时 enabled(否则 disabled/隐藏);→接口 = 04 的端点;空动作=纯前端。
