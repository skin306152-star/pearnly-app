คู่มือ: ส่งข้อมูลจาก Pearnly เข้า MR.ERP 教程:把数据从 Pearnly 送进 MR.ERP

เนื่องจาก MR.ERP ไม่มี API ให้เชื่อมต่อโดยตรง การนำข้อมูลเข้า MR.ERP จึงทำผ่าน ไฟล์ Excel ตามรูปแบบของ MR.ERP ที่ Pearnly สร้างให้ แล้วอัปโหลดเข้าหน้า "นำเข้าข้อมูล" ของ MR.ERP มี 2 วิธี ให้เลือก — ทั้งสองวิธีใช้ไฟล์รูปแบบเดียวกัน ต่างกันแค่ "ใครเป็นคนอัปโหลด"
MR.ERP 没有 API,所以数据进 MR.ERP 都是靠 Pearnly 生成 MR.ERP 格式的 Excel,再传进 MR.ERP 的「导入数据」页。有 2 种方式,用的是同一种文件,区别只在"谁来上传"。

เลือกวิธีไหนดี? / 选哪种?

วิธีที่ 1 · ให้ Pearnly ส่งให้อัตโนมัติ 让 Pearnly 自动送

Pearnly มี "หุ่นยนต์" ล็อกอิน MR.ERP และอัปโหลดให้เอง คุณแค่กดปุ่ม (หรือเปิดให้ส่งอัตโนมัติ)
Pearnly 用"机器人"自动登录 MR.ERP 上传,你只需点一下(或开自动)。
เหมาะกับ: อยากสะดวกที่สุด · ยอมให้ระบบเก็บรหัส MR.ERP ไว้
适合:想最省事 · 愿意让系统保存 MR.ERP 密码

วิธีที่ 2 · ดาวน์โหลดไฟล์ไปอัปโหลดเอง 下载文件自己传

Pearnly สร้างไฟล์ Excel ให้ดาวน์โหลด แล้วคุณนำไปอัปโหลดที่หน้านำเข้าของ MR.ERP เอง
Pearnly 给你下载 Excel,你自己拿去 MR.ERP 导入页上传。
เหมาะกับ: ไม่อยากให้ระบบเก็บรหัส MR.ERP · อยากตรวจก่อนนำเข้าเอง
适合:不想让系统存 MR.ERP 密码 · 想自己先核对再导入

เตรียมก่อนเริ่ม (ทั้งสองวิธีต้องมี) / 开始前的准备(两种都需要)

  • ตั้งค่าการเชื่อมต่อ MR.ERP แล้ว — ชื่อผู้ใช้ / รหัสผ่าน / เลือกบริษัท (เฉพาะวิธีที่ 1 ต้องกรอกรหัสในระบบ)
    已配置 MR.ERP 连接 — 账号/密码/选公司(只有方式1需要在系统里填密码)
  • ลูกค้าในใบกำกับมีรหัสลูกค้าใน MR.ERP แล้ว (หรือเปิด "สร้างลูกค้าอัตโนมัติ") — ไม่งั้น MR.ERP จะไม่รู้จักลูกค้า
    发票客户在 MR.ERP 已有客户码(或开"自动建客户")—— 否则 MR.ERP 认不出客户
  • ใบกำกับข้อมูลครบ — ระบุลูกค้าแล้ว, มีเลขที่เอกสาร, วันที่, ยอดเงิน
    发票字段齐全 —— 已分配客户、有发票号、日期、金额

วิธีที่ 1 · ให้ Pearnly ส่งให้อัตโนมัติ / 方式1:让 Pearnly 自动推送

Pearnly จะใช้ระบบอัตโนมัติ (Playwright) ล็อกอิน MR.ERP, สร้างไฟล์ Excel, อัปโหลด และยืนยันให้เองทั้งหมด · Pearnly 用自动化程序(Playwright)替你登录、生成 Excel、上传、确认,全程自动。

1.1 ตั้งค่าการเชื่อมต่อ MR.ERP (ทำครั้งเดียว) 配置 MR.ERP 连接(只做一次)

1
เปิดเมนู "การเชื่อมต่อ (Integration)" ที่แถบซ้าย → การ์ด "เชื่อมต่อ ERP"
点左侧栏「集成」→ 找到「ERP 对接」卡片
แถบซ้ายล่าง 集成 / Integration → ส่วน ERP 系统 → การ์ด ERP 对接 → ปุ่ม 配置 (ตั้งค่า)
2
กด "เพิ่ม Endpoint" แล้วเลือก MR.ERP · กรอกชื่อผู้ใช้ / รหัสผ่าน / เลือกบริษัท · กด "ทดสอบการเชื่อมต่อ" แล้วบันทึก
点「新增端点」选 MR.ERP · 填账号/密码/选公司 · 点「测试连接」后保存
ปุ่ม 新增端点 (เพิ่ม Endpoint) · กรอกข้อมูล · ทดสอบการเชื่อมต่อ → บันทึก

1.2 ส่งข้อมูล — มี 2 แบบ 推送 —— 两种触发

3
แบบกดเอง: เปิดใบกำกับที่อ่านเสร็จ → กดปุ่ม "ส่งเข้า ERP" มุมขวาบน
手动:打开识别好的发票 → 右上角点「推送到 ERP」
ในหน้ารายละเอียดใบกำกับ มุมขวาบน · ปุ่ม ส่งเข้า ERP (推送到 ERP) · ถ้ามีหลาย ERP จะให้เลือกปลายทาง
4
แบบอัตโนมัติ: เปิดสวิตช์ไว้ พออ่านใบกำกับเสร็จระบบส่งให้เลย ไม่ต้องกด
自动:开开关后,识别完自动送,不用点
ตั้งที่ 集成 → ERP 对接 · สวิตช์ 识别成功后自动推送 (ส่งอัตโนมัติหลังอ่านเสร็จ) ของแต่ละ endpoint + โหมดด้านบน (智能分拣 / 固定账套 / 只识别不推送)
ℹ️ หลังกดส่ง หุ่นยนต์จะทำให้เอง: ล็อกอิน MR.ERP → สร้าง Excel → อัปโหลดหน้านำเข้า → ยืนยัน → อ่านผลกลับมา · ดูผลได้ที่ 集成 → 推送日志 (บันทึกการส่ง) ว่าสำเร็จ/ล้มเหลว
点了之后机器人自动:登录→生成Excel→上传导入页→确认→读回结果 · 在「集成→推送日志」看成功/失败

วิธีที่ 2 · ดาวน์โหลดไฟล์แล้วนำเข้า MR.ERP เอง / 方式2:下载文件自己导入 MR.ERP

เหมาะถ้าไม่อยากเก็บรหัส MR.ERP ไว้ในระบบ · Pearnly สร้างไฟล์ให้ คุณนำไปอัปโหลดเองที่ MR.ERP · 不想把 MR.ERP 密码存系统时用 · Pearnly 出文件,你自己上传。

1
ในหน้ารายละเอียดใบกำกับ กดปุ่ม "ดาวน์โหลดไฟล์ MR.ERP" ใหม่ / 新
在发票详情页,点「下载 MR.ERP 表格」按钮
ปุ่ม 下载 MR.ERP 表格 (ดาวน์โหลดไฟล์ MR.ERP) · อยู่ข้างปุ่ม "ส่งเข้า ERP" · ได้ไฟล์ Excel รูปแบบ MR.ERP มา 1 ไฟล์
⚠️ ถ้าใบกำกับยังไม่พร้อม (เช่น ลูกค้ายังไม่มีรหัสใน MR.ERP) ระบบจะบอกว่าขาดอะไร — แก้ให้ครบก่อนถึงจะดาวน์โหลดได้
若发票没准备好(如客户没 MR.ERP 码),系统会提示缺什么,补齐才能下载。
2
ล็อกอินเข้า MR.ERP ด้วยบัญชีของคุณ
用你的账号登录 MR.ERP
3
เข้าเมนู "นำเข้าข้อมูล" → เลือกประเภท "ขายเชื่อ-รายได้ขายในประเทศ"
进「数据导入(นำเข้าข้อมูล)」菜单 → 类型选「赊销-国内销售收入」
เมนูบนสุด นำเข้าข้อมูล → หน้า "นำเข้าขายสินค้า"
4
กด "เลือกไฟล์" → เลือกไฟล์ที่ดาวน์โหลดจาก Pearnly → กด "อัพโหลด"
点「选择文件」→ 选刚从 Pearnly 下载的文件 → 点「上传」
5
ตรวจรายการในหน้าพรีวิว → ติ๊กเลือก → กด "นำเข้าข้อมูล" เพื่อยืนยัน
在预览页核对 → 勾选 → 点「นำเข้าข้อมูล(导入)」确认
หน้าพรีวิวจะแสดง เลขที่ / วันที่ / รหัสลูกค้า / เลขที่บิล ให้ตรวจก่อนยืนยัน

ถ้าส่ง/นำเข้าไม่สำเร็จ / 如果送不进去

อาการ / 情况สาเหตุ + วิธีแก้ / 原因+怎么办
รหัสผ่านผิด
密码错		 ชื่อผู้ใช้/รหัส MR.ERP ผิด หรือยังไม่กรอก → ไปแก้ที่หน้าตั้งค่า ERP / 去 ERP 设置改账号密码
